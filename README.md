# Assorted AI Agents & Skills

## Agents
- waterfall-architect
- waterfall-planner
- waterfall-tester

## Skills
- commit-message
  - Commit message formatting
- git-commit
  - Breaking changes down into one or more commits
- structured-logging
  - log message formatting inspired from "loggingsucks.com"
- software-architecture
  - Contrary to the name, this skill is more for keeping up to date documentation about project architecture.
- feature-file
  - Implementation of the feature list from "Effective Harnesses for Long-Running Agents" by Anthropic.
  - Manages a persistent artifact tracking project features and implementation status
- waterfall-development
  - Builds on the feature-file for structured development processes across multiple agent sessions.
